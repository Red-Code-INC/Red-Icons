About

What Is Red Icons

Red Icons is an Icon Library (like Font Awesome and Bootstrap Icons), this was originally going to be a function of Red CSS but we decided it would be better to separate the icons from it.

Red Icons has also replaced Red Logos meaning you can also access pure logos of the Red Code Family

All you have to do is add <i class="rc-name"></i>. basically the logos are just like the icons but instead of ri its rc and sizing uses width instead of font-size


How To Add The Icons

Its really simple, there are 2 ways to do it, you can either add an html element with the class ri-insertnameoficonhere or you can make an element's font-family be Red-Icons and type the name of the icon you want

NOTE: only the first option works for the Red Code Family logos (the ones with the rc instead of ri)
